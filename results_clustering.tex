\subsubsection{Clustering Coefficient}
Figure \ref{fig:cluster_coeff} shows the average clustering coefficient and the average shortest path length for different $p$, normalised by the value at $p=0$. 
For the values $p=0.03$ and $p=0.1$ we can observe the small-world effect. 
The clustering coefficient retains relatively high values at around 0.9 for $p=0.03$ and 0.75 for $p=0.1$. 
At the same time, the average shortest path length has dropped to 0.4 and 0.25 of the value at $p=0$. 
Thus, we can still see a substantial clustering while the average shortest path length has already decreased.
