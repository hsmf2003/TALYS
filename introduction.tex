\section{Problem Statement}

In this lab report two topics are studied: Watts-Strogatz networks and Markov-chain convergence. 

A Watts-Strogatz network is a type of random undirected graph (a collection of nodes and connecting vertices) that is used to model social networks. In the first part of this report Watts-Strogatz networks are studied using large statistical sets and then varying different parameters of the network. The studied network characteristics are the node degree distributions, statistics of connected components, shortest path distance, diameter, average path length and the clustering coefficient. 

A Markov chain is a stochastic process that describes a sequence of possible events in which the probability of each event depends only on the state attained in the previous event. In the second part of this report Markov-chain convergence to its stationary distribution is studied. First the theory behind  Markov chain convergence, stationary distribution, error dynamics and spectral decomposition is explained. Then the convergence to a stationary distribution is numerically computed.

Based on the findings of the two topics' questions about both Watts-Strogatz networks and Markov chains are answered in the Synthesis section of this report.
