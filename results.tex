\section{Results}

The following figures show that the comparison between TALYS and EXFOR cannot be based only on the visual overlap near the main peak. In several proton-production cases, the four modes are quite close around that area, which makes this region less useful for choosing between them. The differences become clearer in the shoulder and in the high-energy tail. This is exactly where pre-equilibrium emission is expected to be more important, and therefore this region gives more information about the effect of changing \texttt{preeqmode}.
The ratio plots are useful because they make these differences more visible. When the total spectra look similar, the ratio can still show whether one mode is systematically above or below the experimental data. The component plots also help to understand why different regions of the spectrum behave differently.

Table \ref{tab:summary_results} summarizes the obtained data and result interpretation. It shows that there is not one single pre-equilibrium mode that works best for all cases. One interesting pattern, however, is that proton production is generally reproduced more smoothly, while deuteron production shows larger discrepancies and a stronger sensitivity to the cut-off region.
