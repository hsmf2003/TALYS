\subsubsection{Connected components}

To study the connected components of a Watts-Strogatz graph a pyton script \verb+WS_2_connected.py+ was written. It can be run using the command \verb+python3 WS_2_connected.py+. The script creates a Watts-Strogatz graphs with fixed parameters $N=200$ and $k=2$ and a variable parameter $p$ that is varied from $0.0$ to $1.0$. For each $p$ value 1000 separate instances of the graph are created. Then the code logs if the graph is fully connected or not.  For every instance the number of connected components and the size of the largest connected component is recorded. For disconnected graphs the distance quantities shortest path length, radius and diameter are also computed. These quantities are then averaged over the 1000 instances. Finally a graph is created for each quantity showing its relation to $p$. All the graphs are saved in the file \verb+WS_2_connected_plot.png+.
